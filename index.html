Border

Свойство CSS border - это универсальный инструмент для задания границ вокруг элементов на веб-странице. Оно позволяет одновременно указать три ключевых параметра рамки: толщину (border-width), стиль (border-style) и цвет (border-color). Граница визуально отделяет содержимое элемента от окружающего пространства, создавая чёткие границы и улучшая восприятие структуры страницы.

border: 5px #fff solid;
border: 5px red dotted;
border: 5px #00ff4c dashed;
border: 10px rgb(145, 113, 76) double;
border: 10px rgba(45, 156, 198, 0.3) groove;
border: 10px rgba(45, 156, 198, 0.9) ridge;
border: 10px aqua inset
border: 10px #e100ff outset;

Border-radius

Свойство CSS border-radius отвечает за скругление углов рамки и фона элемента, позволяя создавать плавные закругления вместо острых углов.

border-radius: 15px;
border-radius: 15px 100px;
border-radius: 100px 15px;
border-radius: 100px 100px 15px 15px;
border-radius: 15px 15px 100px 100px;
border-radius: 15px 30px 60px 90px;
border-radius: 50%;
border-radius: 15% 100%;

Border-top/right/bottom/left

border-top: 15px rgba(45, 156, 198, 0.9) ridge;

border-left: 15px rgba(45, 156, 198, 0.9) dotted;

Border-image/collapse/spacing

Border-image

Свойство border-image позволяет использовать изображение вместо стандартной рамки вокруг элемента.

border-top: 5px #fff solid;
border-bottom: 5px red dotted;
border-left: 5px #00ff4c dashed;
border-right: 10px rgb(145, 113, 76) double;
border-top: 10px rgba(45, 156, 198, 0.3) groove;
border-top: 15px rgba(45, 156, 198, 0.9) ridge;
border-top: 10px aqua inset;;
border-top: 15px red dotted;

Border-collapse

Свойство border-collapse применяется к таблицам и управляет тем, как обрабатываются границы ячеек.

Border-spacing

Свойство border-spacing задаёт расстояние между границами ячеек таблицы при border-collapse: separate.
